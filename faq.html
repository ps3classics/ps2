Frequently Asked Questions

What is this site?

This website aims to be the definitive resource for PS2 emulation on PS3.
Our focus is "PS2 Classics", the software emulator available for all PS3 models.

Where does the data come from?

Compatibility info is currently obtained from the PS3 Developer Wiki compatibility list.

Does this emulator work on any PS3?

Yes. Sony developed a PS2 emulator for their digitally released "PS2 Classics" called "ps2_netemu". It is entirely software emulated and works on any model. Technical information and comparisons are also available on PS3 Developer Wiki.

How can I play PS2 games on any PS3?

If you have Cobra CFW, you can simply place your ISO into the "PS2ISO" folder at the root of the HDD. Otherwise, install the game as a PKG. There is no compatibility difference as they both use netemu.

The folks at PS2 Classics Vault host hundreds of PS2 Classics, curated by hand and verified to work on any PS3.

How can I contribute?

If a game works perfectly, you may edit the Wiki directly.
If you test an untested game and discover it has issues, add it to the Wiki and open an issue via the GitHub repository.
